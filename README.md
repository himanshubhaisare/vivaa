# vivaa
Coding Exercise

Steps taken

Installed Homebrew
`https://brew.sh/`

Installed Node/NPM
`brew install node`

created directory for server code
`cd vivaa`

created the server index file
`vi index.js`

Init package manager
`npm init`

Installed express 
`npm install express --save`

Start the server
`node index.js`

The application runs at 
`http://localhost:4000/`
